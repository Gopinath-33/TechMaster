# TechMaster

An online learning platform built with HTML, CSS, and JavaScript — inspired by Udemy. TechMaster allows users to browse, explore, and enroll in a wide range of tech courses at affordable prices.

🌐 **Live Demo:** [https://gopinath-33.github.io/TechMaster/](https://gopinath-33.github.io/TechMaster/)

---

## 📌 Features

- 🏠 **Hero Section** — Promotional banner with flash sale offer and background video
- 📚 **Course Listings** — Cards with course image, title, instructor, rating, and discounted price
- 🔥 **Popular Topics** — Quick-access tags (Python, React JS, Data Science, ML, etc.)
- 📊 **Stats Section** — Platform highlights (15K+ Courses, 500K+ Students, 2K+ Instructors, 100+ Countries)
- 📩 **Enroll Now** — Links to contact/enrollment page
